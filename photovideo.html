イベントなどの模様を紹介した写真や動画

イベント写真

　

これらは2013年の第1回会議の模様です。第2回の写真はにシェアされています。

  • 各団体の発表

    これまで復興支援に関わって来た各種団体による活動報告。写真は石巻でITエンジニア育成を目指すイトナブ古山さんによる報告の様子。

  • 震災当時の情報経路

    震災当時、被災地で有効だった情報の入手先がiSPPより報告されました。ネットよりもラジオが震災直後に大きく活躍した様子が分かります。

  • 全日本芋煮会同好会

    2013年10月6日のイベントでは、全日本芋煮会同好会のみなさんが早い段階から、参加者に向けて芋煮の準備をしてくださいました。

  • テーマの洗い出し

    アンカンファレンスでは議論したいテーマを参加者自ら提案していきます。それぞれの提案をカテゴライズして集約、チーム分けをします。

  • 芋煮会の様子

    芋煮同好会によって振る舞われた芋煮は参加者の交流を促し一体感を生み、その後のアンカンファレンスでの活発な議論につながりました。

  • テーマ発表

    参加者から提案された議題が15のテーマにカテゴライズされ、3回のセッションで興味のある人でグループになり話し合われました。

  • テーマ選択

    参加者はカテゴライズされたテーマの中から、自分が興味を持てるテーマを探して、そのグループに入ってそのテーマを議論します。

  • アンカンファレンス

    テーマに関するトピックリーダーとファシリテーターが上手く連携して、グループに参加した人の意見をまとめあげていきます。

  • 参加者の様子

    「IT x 災害」会議の参加者は国内外、さまざまな背景を持ちつつも同じ被災地支援活動に関わった人達で構成されています。

  • Wrap Up

    Wrap Upの時間では、各グループで議論された内容をトピックリーダーが報告し、他グループの人達に内容を共有しました。

  • 継続した活動へ

    2013年10月6日のイベントでは、それぞれが独自に活動していたコミュニティが繋がり、継続した活動となるよう会の終わりに再確認されました。

  • イベント終了後の懇親会

    今回のイベントを終えて、参加者の交流をより促すために懇親会が行われました。コミュニケーションを深めていくことが大切です。

イベント動画

これらは2013年の第1回会議の模様です。第2回の動画は準備中です。

  • 「IT x 災害」会議 午前の部

    開会の挨拶、および、各団体からのショートスピーチによる活動の報告、活動における課題の共有を行いました。

  • 「IT x 災害」会議 芋煮会

    全日本芋煮会同好会の皆さんから、山形風芋煮が参加者に振る舞われました。温かい芋煮のおかげで交流が深まりました。

  • 「IT x 災害」会議 午後の部

    午前中に参加者の皆さんが提出した議題から生まれたテーマを元にグループに分かれてアンカンファレンスが実施されました。

  • 「IT x 災害」会議 まとめ

    各グループのトピックリーダーから、アンカンファレンスで話し合われた結果を参加者全員に共有し、「IT x 災害」会議を終えました。

概要

東日本大震災から3年半が過ぎました。その間、私たちはITやインターネットの力を使い支援活動に取り組んできました。復興への関心の低下が指摘される中、復興はいまだ緒に就いたばかりです。地域の生活や人々のつながりを新たに築くには、まだまだ時間がかかります。また復興の先にある地域振興やまた起こると予想される災害への備えも大きな課題です。
